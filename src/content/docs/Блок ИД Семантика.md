Императивно-Временная Функция - Интровертированная Интуиция (и). 

Эта функция императивно, неотвратимо, автоматически (и) погружает ТИМ в активность, в динамичность (и) среды. Эта функция является обратной стороной Программной. В связи с этим она является самой не осознанной, подсознательной (и). Она формирует прогноз (и) деятельности для реализации идей. Информация по аспекту этой функции вводит ТИМ в рабочий режим (и + Л), вдохновляет (и), создает особое творческое состояние (и), создает чувство перспективы (и). Тематику этой функции ТИМ использует в качестве универсального критерия разумности, целесообразности (и) и оценивает по этому критерию поступки других, их смысл (и). Эта функция автоматически включается при необходимости помощи по тематике аспекта, который она обрабатывает. В сосредоточенном (и) состоянии (и) Субъект много внимания уделяет теме этой функции.
