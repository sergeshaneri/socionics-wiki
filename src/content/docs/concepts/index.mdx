# Core Concepts

Socionics is built upon several fundamental concepts that explain how personality types process information and interact with each other.

## Information Elements

The foundation of socionics is the theory of information elements (also called IM elements). There are 8 information elements, each representing a different way of perceiving and processing information:

- **Te** (Extroverted Thinking) - External logic, efficiency, productivity
- **Ti** (Introverted Thinking) - Internal logic, structure, classification
- **Fe** (Extroverted Feeling) - External emotions, social atmosphere
- **Fi** (Introverted Feeling) - Internal emotions, personal values, relationships
- **Se** (Extroverted Sensing) - External sensory perception, force, will
- **Si** (Introverted Sensing) - Internal sensory perception, comfort, aesthetics
- **Ne** (Extroverted Intuition) - External intuition, possibilities, potential
- **Ni** (Introverted Intuition) - Internal intuition, time, patterns, foresight

## Model A

Model A is the structural representation of how information elements are arranged in each personality type. It consists of 8 positions that show the strength and conscious awareness of each information element for a given type.

## Intertype Relations

Socionics describes 16 different types of relationships between personality types, ranging from highly compatible (dual, activator) to potentially problematic (conflict, supervision). These relations help predict compatibility and potential conflicts in interpersonal relationships.
