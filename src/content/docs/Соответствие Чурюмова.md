Аспектон

  

Принцип Полисемантичности (ППС) проявляется в Аспектоне в соответствии: Инорфмационный Аспект - Соционическая Функция - Признак Функции.

1. Единичный Элемент - Экстравертированная (Черная) Интуиция - Программно-Потенциальная (Базовая) Функция - Существование.
    
2. Интровертированная (Белая) Сенсорика - Экспектационно-Ресурсная (Суггестивная) Функция - Экстравертные/Интровертные (Объектные/Релятивные). Этот признак имеет схожую семантику с признаком Инф2, благодаря пониманию Информационных Потоков можно понять, как это проявляется в каждой СФ.
    
3. Экстравертированное Эмоционирование (ЧЭ) - Мотивационно-Мобилизационная Функция (Активационная) - Оценочные/Ситуативные.
    
4. Интровертированное Мышление (БЛ) - Коммуникативно-Зондирующая Функция (Творческая) - Программные/Потенциальные (Сильные/Слабые).
    
5. Экстравертированное Мышление - Реализационно-Технологическая Функция (Фоновая) - Вербальные/Лаборные (Ценностные/Неценностные).
    
6. Интровертированное Эмоционирование - Консервативно-Критическая Функция (Болевая) - Инертные/Контактные.
    
7. Экстравертированная Сенсорика - Адаптивно-Нагрузочная Функция (Ролевая) - Акцептные/Продуктивные. Этот признак подобен по семантике предпоследнему признаку Инф2.
    
8. Интровертированная Интуиция - Императивно-Временная Функция (Ограничительная) - Ментальные/Витальные. Этот признак подобен по семантике последнему признаку Инф2.
    

  

Соответствие дихотомий:

  

Экстравертные/Интровертные = Выразительные/Впечатлительные

  

Оценочные/Ситуативные = Инерционные/Двигательные

  

Сильные/Слабые = Отвлеченные/Вовлеченные

  

Вербальные/Лаборные = Целеполагающие/Исполнительные

  

Инертные/Контактные = Скрытые/Явные

  

Акцептные/Продуктивные = Дискретные/Континуальные

  

Ментальные/Витальные = Статичные/Динамичные
